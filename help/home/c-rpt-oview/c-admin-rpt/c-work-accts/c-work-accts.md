# 使用帳戶{#working-with-accounts}

{{eol}}

使用個別使用者和群組帳戶控制Report Portal中的存取權和權限。

某些使用者可指定為管理員，以管理所有帳戶。

個別使用者和群組帳戶都指定特定使用者或群組可以存取的設定檔和索引標籤。 主要差異在於個別帳戶可讓使用者重設其密碼，而群組帳戶密碼只能由 [!DNL Report Portal] 管理員。

以下各節提供您檢視、建立及編輯個別使用者和群組帳戶所需的資訊。
