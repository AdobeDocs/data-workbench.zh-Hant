# 使用工作區窗口菜單{#using-the-workspace-window-menu}

此功能表可讓您將可用的視覺化新增至工作區。

在工作區視窗（位於工作區背景的任何位置）中按一下滑鼠右鍵。

![](assets/mnu_workspace.png)

>[!NOTE]
>
>由於您可完全自訂資料工作台的實施，所以顯示的功能表項目可能與本節所述內容不同。
