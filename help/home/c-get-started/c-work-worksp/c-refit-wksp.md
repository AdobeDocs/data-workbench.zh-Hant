# 重新調整工作區{#refit-a-workspace}

{{eol}}

重新調整工作區會根據您使用的顯示解析度調整視覺效果大小。

在工作區中，按一下 **[!UICONTROL File]** > **[!UICONTROL Refit Workspace]**.

這有助於快速調整演示文稿的顯示。
