# 處理圖例{#processing-legend}

「處理圖例」提供有關特定伺服器的資料處理和轉換的詳細資訊，使您能夠跟蹤正在重新處理和重新轉換的資料的進度。

![](assets/vis_ProcessingLegend.png)

  </tr> 
  <tr>
