# 將檔案複製並貼為附件{#copy-and-paste-a-file-as-an-attachment}

在Microsoft Outlook中複製檔案並貼上為電子郵件附件的步驟。

* 按一下右鍵檔案的複選標籤，然後按一下 **[!UICONTROL Copy]**。 在電子郵件正文中，按Ctrl+v以附加檔案。
