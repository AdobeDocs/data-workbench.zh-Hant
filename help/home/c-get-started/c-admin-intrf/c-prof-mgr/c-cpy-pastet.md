# 複製並貼上檔案以作為附件{#copy-and-paste-a-file-as-an-attachment}

{{eol}}

在Microsoft Outlook中複製和貼上檔案作為電子郵件附件的步驟。

* 以滑鼠右鍵按一下檔案的勾號，然後按一下 **[!UICONTROL Copy]**. 在電子郵件內文中，按Ctrl+v附加檔案。
