# 在地圖中搜尋{#search-within-a-map}

搜索包含特定文本的元件的步驟。

1. 在相依性映射中按一下右鍵，然後按一下 **[!UICONTROL Search]**。
1. 在文字方塊中，輸入要搜尋的文字。

   ![步驟資訊](assets/vis_DependencyMap_Search.png)

地圖會反白顯示名稱或參數設定包含搜尋文字之元件的節點。
