# 在地圖中搜尋{#search-within-a-map}

{{eol}}

搜尋包含特定文字之元件的步驟。

1. 在相依性對應內按一下滑鼠右鍵，然後按一下 **[!UICONTROL Search]**.
1. 在文字方塊中，輸入要搜尋的文字。

   ![步驟資訊](assets/vis_DependencyMap_Search.png)

地圖會反白標示名稱或參數設定包含搜尋文字之元件的節點。
