# 顯示突出顯示的節點{#display-a-highlighted-node}

有關在另一個相關性映射中顯示突出顯示節點的資訊。

如果相關性路徑在相關性映射中突出顯示，您可以選擇只顯示其他映射中的那些節點。

* 按一下右鍵從屬關係路徑中的一個節點，然後按一下 **[!UICONTROL Show Only These Nodes]**。

   新映射在註解中開啟，當按一下右鍵註解中的元件時，所有其它菜單選項都可用。
