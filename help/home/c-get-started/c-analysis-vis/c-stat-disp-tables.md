# 表格中的統計顯示{#statistical-display-in-tables}

統計顯示可讓您分析多維表格中資料彼此的關聯情形。

例如，您可能想要知道購買紅鞋是否與購買紅帶或不同顏色帶有統計上的關聯。 統計資料會顯示您的關係是高於預期，還是低於預期。

在表格的上邊框上，按一下滑鼠右鍵 **[!UICONTROL Display As]** > **[!UICONTROL Stats]**。
