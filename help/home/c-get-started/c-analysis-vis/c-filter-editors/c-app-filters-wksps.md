# 將篩選器套用至工作區{#apply-a-filter-to-a-workspace}

{{eol}}

將篩選器套用至工作區的步驟。

* 在篩選器編輯器視窗中，按一下 **[!UICONTROL Apply Filter]**.

   篩選器編輯器視窗的邊框會變黃色，表示篩選器已將選取項目套用至工作區。
