# 將篩選套用至工作區{#apply-a-filter-to-a-workspace}

將篩選器套用至工作區的步驟。

* 在篩選編輯器視窗中，按一下 **[!UICONTROL Apply Filter]**。

   篩選器編輯器窗口的邊框變為黃色，表示篩選器已將選項應用於工作區。
