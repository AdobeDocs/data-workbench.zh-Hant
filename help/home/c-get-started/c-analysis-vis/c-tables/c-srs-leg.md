# 啟用序列圖例{#enable-a-series-legend}

維度中的元素可以以系列的方式進行色彩編碼，讓每個元素在工作區中的其他視覺化中都能更容易追蹤。

**啟用或禁用系列圖例**

* 按一下右鍵要編寫顏色代碼的維的元素或標籤，然後按一下&#x200B;**[!UICONTROL Series Legend]**。

   當系列圖例處於活動狀態時，菜單選項旁會出現X，尺寸內的元素以交替顏色顯示。

   ![](assets/vis_Graph_SeriesLegend.png)
