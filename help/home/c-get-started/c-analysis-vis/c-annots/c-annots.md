# Annotations{#annotations}

註解是使用者可新增至工作區的視窗，以包含描述性註解、圖形或動態標題。

註解也可以新增至工作區做為註解。 有關使用注釋註解的資訊，請參 [閱將註解添加到工作區](../../../../home/c-get-started/c-vis/c-call-wkspc.md#concept-212b09e763044d938987b4a9c658adc0)。
