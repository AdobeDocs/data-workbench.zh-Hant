# 區段{#segments}

{{eol}}

區段可讓您從選取項目即時建立維度。

區段在處理您想要作為維度的不相連、非重疊的群組時（例如根據先前的購買比較兩種類型的購物者）最有用。

使用區段只會影響您使用區段維度的工作區。
