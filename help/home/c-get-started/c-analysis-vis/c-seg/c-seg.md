# 區段{#segments}

區段可讓您從選取範圍即時建立維度。

使用您想要當做維度使用的脫節、非重疊的群組時，例如根據先前購買比較兩種類型的購物者時，區段最有用。

使用區段只會影響您使用區段維度的工作區。
