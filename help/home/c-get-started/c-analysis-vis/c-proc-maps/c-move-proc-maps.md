# 移動和旋轉程序圖{#move-and-rotate-a-process-map}

{{eol}}

有關在窗口內移動工藝圖以及如何旋轉3D工藝圖的資訊。

**在窗口內移動進程映射**

* 按一下「 」，然後將地圖拖曳至所需位置。

**旋轉3D工藝圖**

* 按一下右鍵並拖動地圖，同時按一下滑鼠按所需的旋轉方向。
