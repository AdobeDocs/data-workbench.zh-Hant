# 移動和旋轉流程圖{#move-and-rotate-a-process-map}

有關在窗口中移動流程圖以及如何旋轉3D流程圖的資訊。

**在窗口內移動進程映射**

* 按一下並拖曳地圖至所需位置。

**旋轉3D工藝圖**

* 在將滑鼠移至所需旋轉方向時，以滑鼠右鍵按一下並拖曳地圖。
