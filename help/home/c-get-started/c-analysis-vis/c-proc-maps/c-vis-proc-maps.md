# 從程序圖中建立視覺效果{#create-a-visualization-from-a-process-map}

您可以從程式圖中建立其他視覺效果。

例如，您可以建立圖形、表格或散布圖。

* 按一下右鍵任何節點，然後按一下&#x200B;**[!UICONTROL Add Visualization]** > *&lt;**[!UICONTROL visualization type]**>*。

程式圖上的每個節點都會成為「地圖」維度的元素。 您可以使用選取的視覺效果功能來分析節點資料。
