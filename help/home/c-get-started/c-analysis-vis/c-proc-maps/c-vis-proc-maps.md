# 從程序圖中建立視覺效果{#create-a-visualization-from-a-process-map}

您可以從流程圖建立其他視覺化。

例如，您可以建立圖形、表格或散布圖。

* 按一下右鍵任意節點，然後按一下&#x200B;**[!UICONTROL Add Visualization]** > ***[!UICONTROL visualization type]***。

進程映射上的每個節點都將成為映射維的元素。 您可以使用選取的視覺化功能來分析節點資料。
