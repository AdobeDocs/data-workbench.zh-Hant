# 移動節點{#move-a-node}

只要映射具有相同的基本維，您就可以在映射內或進程映射之間移動節點。

例如，可以將節點從3D進程映射拖動到2D進程映射。

* 按一下「 」，然後將節點拖曳至相同程式圖中或工作區中其他程式圖中的所需位置。
