# 新增圖說文字{#add-a-callout}

{{eol}}

有關將註解添加到2D進程圖的資訊。

以滑鼠右鍵按一下節點，然後按一下 **[!UICONTROL Add Callout]** > *&lt;**[!UICONTROL callout type]**>*.

如需圖說文字的詳細資訊，請參閱 [將圖說文字新增至工作區](../../../../home/c-get-started/c-vis/c-call-wkspc.md#concept-212b09e763044d938987b4a9c658adc0).

>[!NOTE]
>
>您只能將圖說文字新增至2D程式圖。
