# 新增圖說文字{#add-a-callout}

有關將註解添加到2D進程映射的資訊。

按一下右鍵節點，然後按一下&#x200B;**[!UICONTROL Add Callout]** > ***[!UICONTROL callout type]**>*。

有關圖說的詳細資訊，請參閱[將圖說添加到工作區](../../../../home/c-get-started/c-vis/c-call-wkspc.md#concept-212b09e763044d938987b4a9c658adc0)。

>[!NOTE]
>
>您只能將註解新增至2D流程地圖。
