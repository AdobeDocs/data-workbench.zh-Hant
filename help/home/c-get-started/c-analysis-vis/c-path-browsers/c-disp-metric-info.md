# 顯示量度資訊{#display-metric-information}

{{eol}}

在路徑瀏覽器中顯示度量資訊的相關資訊。

以滑鼠右鍵按一下元素名稱，即可在功能表的頂端區段檢視路徑瀏覽器之關聯量度的數量和百分比。

例如，若您正在處理網站資料，請以滑鼠右鍵按一下頁面名稱，即可檢視功能表頂端區段中「頁面檢視」的數量和百分比。
