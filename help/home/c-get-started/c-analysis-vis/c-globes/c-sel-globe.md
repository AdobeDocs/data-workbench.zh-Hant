# 在地球上進行選取{#make-selections-on-a-globe}

當您從特定圖層在球體上進行選取時，所有其他圖層都只會顯示與選取資料相關的資料。

您可以一起使用多個球，並排顯示不同的圖層子集。 請參閱[在視覺化中選取](../../../../home/c-get-started/c-vis/c-sel-vis/c-sel-vis.md#concept-012870ec22c7476e9afbf3b8b2515746)。
