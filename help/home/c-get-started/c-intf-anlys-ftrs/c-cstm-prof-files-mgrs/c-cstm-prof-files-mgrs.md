# 自訂描述檔管理員和伺服器檔案管理員{#customize-the-profile-manager-and-server-file-manager}

配置檔案管理器和伺服器檔案管理器顯示大量目錄和檔案。

在某些情況下，您可能希望訪問其中一個管理器的子目錄，而不需要導航其整個目錄結構。 要執行此操作，可以建立只顯示所選目錄的新管理器。
