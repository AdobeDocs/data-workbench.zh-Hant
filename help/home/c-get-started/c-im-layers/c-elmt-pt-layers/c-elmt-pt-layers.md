# 元素點層{#element-point-layers}

在Data Workbench中，元素點層在球體上為尺寸的每個元素顯示一個點。

全球顯示的點大小是透過在您要繪製其元素的維度上評估指定量度來決定。 因此，較大的量度值會對應全球的較大點。
