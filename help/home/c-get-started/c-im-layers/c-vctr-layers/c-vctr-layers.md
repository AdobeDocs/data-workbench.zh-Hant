# 使用向量層{#working-with-vector-layers}

{{eol}}

在Data Workbench中，向量層顯示地球上的靜態向量資料（線條圖）。

您可以使用向量層來顯示地理區域的輪廓，例如狀態、國家、島嶼或水體。
