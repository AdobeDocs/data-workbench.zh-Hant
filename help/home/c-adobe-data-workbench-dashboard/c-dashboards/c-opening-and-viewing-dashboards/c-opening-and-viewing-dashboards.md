# 開啟和檢視控制面板{#opening-and-viewing-dashboards}

{{eol}}

控制面板的開啟方式有三種：透過快速存取功能表、控制面板瀏覽器，或使用控制面板超連結。 雖然控制面板介面中一次只能開啟一個控制面板，但您可以建立個別的瀏覽器索引標籤或視窗，以同時檢視多個控制面板。
