# 開啟和檢視控制面板{#opening-and-viewing-dashboards}

控制面板可透過下列三種方式之一開啟：透過快速存取功能表、透過「控制面板瀏覽器」，或使用控制面板超連結。 雖然控制面板介面中一次只能開啟一個控制面板，但您可以建立個別的瀏覽器標籤或視窗，以同時檢視多個控制面板。
