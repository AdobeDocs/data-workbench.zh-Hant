# 製作控制面板副本{#making-a-copy-of-a-dashboard}

複製控制面板的步驟。

1. 載入您要製作副本的控制面板。
1. 從控制面板介面頂部的控制面板菜單中選擇&#x200B;**[!UICONTROL Save As…]**。
1. 出現&#x200B;**[!UICONTROL Save Dashboard As…]**&#x200B;窗口時，在新名稱中輸入控制面板副本。
