# 製作控制面板副本{#making-a-copy-of-a-dashboard}

複製控制面板的步驟。

1. 載入您要複製的控制面板。
1. 從控 **[!UICONTROL Save As…]** 制面板介面頂端的控制面板選單中選取。
1. 出現窗 **[!UICONTROL Save Dashboard As…]** 口時，輸入控制面板副本的新名稱。
