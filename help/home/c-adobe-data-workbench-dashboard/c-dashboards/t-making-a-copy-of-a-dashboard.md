# 製作控制面板副本{#making-a-copy-of-a-dashboard}

{{eol}}

複製控制面板的步驟。

1. 載入您要製作副本的控制面板。
1. 選擇 **[!UICONTROL Save As…]** 從控制面板介面頂端的控制面板功能表。
1. 當 **[!UICONTROL Save Dashboard As…]** 窗口中，輸入控制面板副本的新名稱。
