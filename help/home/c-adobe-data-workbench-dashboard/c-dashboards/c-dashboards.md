# 控制面板{#dashboards}

控制面板的建立是為了視覺化度量，並提供互動式資料分析功能。 按一下控制面板中的項目，您就可以快速輕鬆地將資料分段，從分析中衍生資訊。 您可建立不限數量的控制面板，以利進行短期即時分析或進行長期控制面板功能。 控制面板可保持私密，或輕鬆與其他控制面板使用者共用。 您也可以從任何控制面板匯出資料，以便用於其他工具，例如Microsoft® Excel™。
