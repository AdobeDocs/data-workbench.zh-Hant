# 移除選取項目{#removing-selections}

{{eol}}

您隨時都可移除選取項目。

只需按一下 **[!UICONTROL X]** 表徵圖在「選擇」菜單中的相應選擇項內。

![](assets/selection_remove.png)

如果視覺效果的選取項目已鎖定，您會看到小掛鎖圖示取代 **[!UICONTROL X]** 表徵圖。 若不先解除鎖定視覺效果，則無法移除鎖定的選取項目。

![](assets/selection_remove_locked.png)
