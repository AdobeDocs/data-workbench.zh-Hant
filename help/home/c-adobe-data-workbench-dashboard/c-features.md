# 功能{#features}

Adobe資料工作台控制面板提供這些功能。

* 對Adobe資料工作台資料集執行即時分析。
* 使用簡單的滑鼠導向介面，深入鑽研並切割資料。
* 建立並儲存多個分析控制面板。
* 與其他人共用分析控制面板。
* 存取其他使用者建立的控制面板。
* 匯出資料以用於其他應用程式，例如Microsoft® Excel®。

本指南提供有關Adobe資料工作台控制面板所提供之功能與使用者介面功能的資訊。 由於您可完全自訂控制面板的實作，因此您的功能與本指南中記載的功能可能有所不同。
