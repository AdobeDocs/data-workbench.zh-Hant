# 功能{#features}

「AdobeData Workbench」控制面板提供下列功能。

* 對AdobeData Workbench資料集執行即時分析。
* 使用簡單的滑鼠驅動介面向下切入並切割資料。
* 建立並保存多個分析控制面板。
* 與其他人共用analytics控制面板。
* 存取其他使用者建立的控制面板。
* 匯出資料以用於其他應用程式，例如Microsoft® Excel®。

本指南提供與AdobeData Workbench控制面板一併提供的功能和使用者介面功能的相關資訊。 由於您可以完全自訂控制面板的實作，因此您的功能可能與本指南中說明的有所不同。
