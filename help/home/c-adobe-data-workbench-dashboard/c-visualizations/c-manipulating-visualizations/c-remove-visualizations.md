# 移除視覺效果{#removing-visualizations}

您可以隨時從控制面板畫布中移除視覺效果。

這是使用視覺效果視窗右上角的&#x200B;**[!UICONTROL X]**&#x200B;圖示來完成。 這會從控制面板畫布中移除它，但除非儲存目前的控制面板，否則不會影響任何已儲存的控制面板版本。 視覺效果中出現的任何選取項目都會從控制面板的選取項目中移除。
