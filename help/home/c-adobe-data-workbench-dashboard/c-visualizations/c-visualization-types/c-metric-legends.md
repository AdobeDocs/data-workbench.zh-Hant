# 量度圖例{#metric-legends}

{{eol}}

量度圖例視覺效果可讓您以表格格式顯示量度總計。 第一欄保有量度名稱，第二欄保有量度值。 捲動條將顯示在無法一次在視窗上顯示所有列的事件中。

無法在量度圖例內進行選取，因為未顯示特定維度值。
