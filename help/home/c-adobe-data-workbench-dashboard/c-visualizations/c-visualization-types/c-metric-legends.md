# 量度圖例{#metric-legends}

度量圖例視覺化可讓您以表格格式顯示度量總計。 第一欄包含量度名稱，而第二欄則包含量度值。 當所有列無法一次顯示在視窗中時，就會顯示捲軸。

無法在量度圖例中進行選取，因為不會顯示特定維度值。
