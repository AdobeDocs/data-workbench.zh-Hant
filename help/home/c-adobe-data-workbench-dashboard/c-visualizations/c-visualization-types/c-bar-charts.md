# 長條圖{#bar-charts}

長條圖視覺化可讓您以視覺化方式呈現度量和維度資料，每個資料元素都代表為圖表上的水準長條。 長條圖允許一次最多在圖表上顯示200個唯一資料元素。 維度值會沿著y軸標示，而度量增量則會沿著x軸標示。 每個維度值的列會以其精確的量度值標示。
