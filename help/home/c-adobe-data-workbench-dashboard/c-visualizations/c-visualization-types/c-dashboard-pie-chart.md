# 圓形圖{#pie-chart}

圓形圖視覺化可讓您顯示有限維度量度值的相對百分比。 圓形圖允許一次最多在圖表上顯示20個唯一資料元素。 圓形圖的每個楔塊代表資料集中的維度及其百分比。 每個百分比會相對於圓形圖中顯示的其他維度的量度值計算，而這不一定是整個資料集。

通過按一下任何特定楔形，可以在圓形圖中進行選擇。
