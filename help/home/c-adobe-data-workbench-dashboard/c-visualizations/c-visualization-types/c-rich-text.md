# RTF 文字{#rich-text}

豐富式文字視覺化可讓您在儀表板中內嵌文字資訊。 這對於提供說明、超連結或指導資訊非常有用。 「富格文本」窗口頂部的文本格式工具欄允許您控制窗口正文中的文本格式。

![](assets/rich_text.png)

若要將文字新增至Rich Text視窗，請將游標置於Rich Text視窗的內文中，然後開始輸入。 要編輯富格文本窗口中的文本，請使用此窗口頂部的文本格式工具欄。
