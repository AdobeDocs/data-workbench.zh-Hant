# 直條圖{#column-charts}

欄圖視覺效果可讓您視覺化量度和維度資料，每個資料元素在圖表上以垂直欄顯示。 欄點陣圖表可一次在圖表上顯示最多200個不重複資料元素。 維度值會沿x軸標示，而量度增量則沿y軸標示。 每個維度值的欄會以其確切的量度值標示。
