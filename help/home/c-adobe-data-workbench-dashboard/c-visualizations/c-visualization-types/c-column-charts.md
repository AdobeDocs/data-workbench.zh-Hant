# 柱圖{#column-charts}

欄圖表視覺化可讓您以圖表上垂直欄的形式呈現每個資料元素的度量和維度資料。 欄點陣圖表允許一次最多在圖表上顯示200個唯一資料元素。 維度值會沿x軸標示，而量度增量則沿y軸標示。 每個維度值的欄會以其精確的度量值標示。
