# 視覺效果類型{#visualization-types}

介紹8種不同的視覺化類型。

在選取視覺化時，請務必仔細考慮您所視覺化的維度和量度，以及如何最有效地視覺化資料。 例如，某些視覺化對一次可顯示的資料量有限制。

有8種不同的視覺化類型可用來視覺化和分析您的資料：長條圖、圓柱圖、折線圖、圓形圖、表格、量度圖例、散布圖和富文本。
