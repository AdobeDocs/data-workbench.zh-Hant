# 刪除設定檔連線{#removing-a-profile-connection}

1. 按一下列末尾的&#x200B;**[!UICONTROL X]**&#x200B;以移除與設定檔的連線。
1. 系統會顯示提示，詢問您是否要刪除設定檔連線。 選擇&#x200B;**[!UICONTROL Yes]**&#x200B;以刪除配置式連接。

   ![](assets/delete_connection.png)

   如果操作成功，您會看到一條消息，指出配置檔案連接已成功刪除。
