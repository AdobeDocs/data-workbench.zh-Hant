# 編輯配置式連接{#editing-a-profile-connection}

1. 按一下要編 ![](assets/edit_icon.png) 輯的截面梁連接行末端的齒輪表徵圖。

   ![](assets/edit_profile_connection.png)

1. 編輯描述檔連線的設定，然後按一下 **[!UICONTROL Update Connection]** 儲存。

   如果操作成功，您將看到一條消息，指示已成功更新配置檔案連接。
