# 安裝必備條件{#installation-prerequisites}

開始安裝之前，請使用具有配置伺服器和安裝新軟體權限的管理員帳戶登錄到Web伺服器。

>[!NOTE]
>
>儀表板系統可以部署在單層（IIS和同一伺服器上的資料庫）中，也可以部署為兩層系統。 本指南未涵蓋的兩層方法中涉及其他配置步驟。
