# 系統日誌{#system-logs}

錯誤和審計日誌可在安裝路徑 **[!UICONTROL Logs]** 的目錄中找到。 這些日誌對於審計儀表板系統和調試您可能遇到的任何問題非常有用。 這些日誌使用伺服器的當前日期儲存在檔案中：

Error logs: [!DNL thinclientweb-yyyy-mm-dd.log]

審核日誌： [!DNL thinclientweb-audit-yyyy-mm-dd.log]
