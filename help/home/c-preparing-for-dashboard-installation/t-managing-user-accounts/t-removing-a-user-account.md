# 移除使用者帳戶{#removing-a-user-account}

1. 按一下 **[!UICONTROL X]** 要從系統中刪除的用戶行末尾的表徵圖。

   會顯示提示，詢問您是否要刪除該用戶。 1.選擇 **[!UICONTROL Yes]** 以刪除用戶。

   如果操作成功，您會看到一條消息，指示用戶已成功刪除。
