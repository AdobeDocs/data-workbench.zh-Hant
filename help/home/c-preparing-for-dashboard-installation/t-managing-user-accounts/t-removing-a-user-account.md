# 移除使用者帳戶{#removing-a-user-account}

1. 按一下要從系統中刪除的用戶的行結尾的&#x200B;**[!UICONTROL X]**&#x200B;表徵圖。

   系統會顯示提示，詢問您是否要刪除該使用者。 1.選擇&#x200B;**[!UICONTROL Yes]**&#x200B;以刪除用戶。

   如果操作成功，您會看到一條消息，指示已成功刪除用戶。
