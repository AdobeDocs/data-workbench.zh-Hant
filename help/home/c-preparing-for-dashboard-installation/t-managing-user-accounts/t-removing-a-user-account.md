# 移除使用者帳戶{#removing-a-user-account}

{{eol}}

1. 按一下 **[!UICONTROL X]** 表徵圖。

   系統會顯示提示，詢問您是否要刪除該使用者。 1.選擇 **[!UICONTROL Yes]** 刪除用戶。

   如果操作成功，您會看到一條消息，指示已成功刪除用戶。
