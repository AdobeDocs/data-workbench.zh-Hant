# 編輯使用者帳戶{#editing-a-user-account}

1. 按一下要編輯的用戶行末尾的齒輪表徵圖![](assets/edit_icon.png)

   ![](assets/edit_user_account.png)

1. 編輯使用者的帳戶設定，然後按一下&#x200B;**[!UICONTROL Save]**。

   如果操作成功，您會看到一條消息，指示用戶組已成功保存。
