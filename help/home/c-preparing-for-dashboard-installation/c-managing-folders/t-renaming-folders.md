# 重新命名資料夾{#renaming-folders}

{{eol}}

1. 按一下右鍵要更名的資料夾，然後選擇 **[!UICONTROL Rename…]**.

   ![](assets/rename.png)

1. 輸入資料夾的新名稱，然後按 **[!UICONTROL Enter]**.

   ![](assets/renamed_folder.png)

   如果操作成功，您會看到提示，指出資料夾已重新命名。
