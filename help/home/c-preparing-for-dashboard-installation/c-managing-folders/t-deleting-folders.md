# 刪除資料夾{#deleting-folders}

1. 按一下右鍵要刪除的資料夾，然後選擇&#x200B;**[!UICONTROL Delete]**。

   ![](assets/delete_folder.png)

   >[!NOTE]
   >
   >您必須先將資料夾空白，才能加以刪除。

1. 系統將提示您確定要刪除該資料夾。
1. 如果操作成功，您還會看到提示，指出該資料夾已被刪除。
