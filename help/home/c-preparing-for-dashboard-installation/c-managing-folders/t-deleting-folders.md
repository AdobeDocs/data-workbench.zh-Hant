# 刪除資料夾{#deleting-folders}

{{eol}}

1. 按一下右鍵要刪除的資料夾並選擇 **[!UICONTROL Delete]**.

   ![](assets/delete_folder.png)

   >[!NOTE]
   >
   >資料夾必須為空，您才能刪除它。

1. 系統會提示您確定要刪除該資料夾。
1. 如果操作成功，您也會看到提示，指出資料夾已刪除。
