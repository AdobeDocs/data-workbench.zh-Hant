# 移除使用者群組{#removing-a-user-group}

1. 按一下 **[!UICONTROL X]** 要從系統中刪除的組行末尾的表徵圖。

   會顯示提示，詢問您是否要刪除組1。 按一下 **[!UICONTROL Yes]**.

   如果操作成功，您將看到一條消息，指出已成功刪除組。
