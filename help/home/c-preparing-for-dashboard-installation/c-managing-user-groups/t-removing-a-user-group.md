# 移除使用者群組{#removing-a-user-group}

1. 按一下要從系統中刪除的組的行末尾的&#x200B;**[!UICONTROL X]**&#x200B;表徵圖。

   系統會顯示提示，詢問您是否要刪除群組1。 按一下 **[!UICONTROL Yes]**。

   如果操作成功，您會看到一條消息，指示已成功刪除組。
