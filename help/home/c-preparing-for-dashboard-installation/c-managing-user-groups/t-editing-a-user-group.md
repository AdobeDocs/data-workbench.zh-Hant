# 編輯使用者群組{#editing-a-user-group}

1. 按一下行末尾的齒輪表徵圖可編輯組。

   ![](assets/edit_user_group.png)

1. 編輯群組的設定，然後按一下&#x200B;**[!UICONTROL Save]**&#x200B;以儲存變更。

   如果操作成功，您會看到一條消息，指示用戶組已成功保存。
