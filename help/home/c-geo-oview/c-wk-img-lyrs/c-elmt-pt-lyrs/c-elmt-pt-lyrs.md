# 使用元素點圖層{#working-with-element-point-layers}

在資料工作台中，元素點圖層會針對維度的每個元素在地球上顯示一個點。

全球顯示的點大小是透過在您要繪製其元素的維度上評估指定量度來決定。 因此，較大的量度值會對應全球的較大點。
