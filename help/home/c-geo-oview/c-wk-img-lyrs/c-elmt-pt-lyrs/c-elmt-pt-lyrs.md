# 使用元素點層{#working-with-element-point-layers}

在Data Workbench中，元素點層會針對一個維度的每個元素在地球上顯示一個點。

全球上顯示的點的大小，是透過在您要繪製其元素的維度上評估指定的量度來決定。 因此，較大的量度值對應於地球上較大的點。
