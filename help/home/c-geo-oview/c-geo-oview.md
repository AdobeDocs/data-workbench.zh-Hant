# Data Workbench地理概觀{#data-workbench-geography-overview}

資料工作台地理位置是地理影像和分析軟體產品，可讓您在資料工作台中顯示地形影像。

如需有關安裝資料工作台[!DNL Geography]的詳細資訊，請參閱[安裝Insight Geography](../../home/c-geo-oview/c-inst-geo/c-inst-geo.md)。

資料工作台[!DNL Geography]包含[!DNL Geography]描述檔，其中包含預設影像圖層及其相關查閱檔案，以及用於分析的其他維度（城市、座標、國家、DMA和網域）。

您也可以新增其他圖層和查閱檔案，以擴充影像功能。 如需影像圖層的詳細資訊，請參閱[使用影像圖層](https://docs.adobe.com/content/help/en/data-workbench/using/client/imagery-layers/c-ustd-img-layers.html)。

如果您有[!DNL Geography]的授權資料工作台，您可以訂閱選用的資料服務，讓您對應全球的IP位址，或建立視覺化，顯示以IP位址為基礎的地理相關資料維度（例如區域代碼、時區和MSA）。 如需訂閱這些服務的詳細資訊，請聯絡Adobe。
