## 升級需求 {#section-84f8f38e9c424d7788c79146fce5a6fe}

升級至Data Workbench6.5時，請遵循這些需求和建議。

* **`Components for Processing Servers\Communications.cfg`**&#x200B;檔案中的變更需要您更新此檔案，以進行DWB 6.5版。 已刪除&#x200B;*SourceListServer*、*SegmentExportServer*&#x200B;和&#x200B;*NormalizeServer*&#x200B;條目。 （DPU不應執行&#x200B;*sourcelist*、*區段匯出*&#x200B;或&#x200B;*標準化伺服器*。）

## 系統更新 {#section-bcdd732af53249b893d30bb42eeb14fc}

已重新命名、刪除下列功能，或者本版本中已重新架構下列功能的安裝檔案或資料夾:

* 「關聯和諧」、「關聯和諧」、「關聯和諧」、「關聯矩陣」、「傾向分數」和「最適歸因」視覺效果現在為&#x200B;**多遍視覺效果**，可讓它們在報表伺服器中執行。

   >[!IMPORTANT]
   >
   >工作區中只能有一個多通視覺效果。
   >
   >工作區中有多個多遍視覺效果時，報表伺服器預設無法產生報表，並擲回下列錯誤：
   >
   >
   ```
   >Too many Multipass visualizations in workspace ....... (has #, 1 allowed).
   >```
   >
   >更新[!DNL ReportServer.cfg]檔案或將此行新增至&#x200B;*Reporting*&#x200B;區段中的現有檔案，以避免發生此錯誤。   >
   >
   >
   ```
   >Max Multipass Per Slice = int: 1
   >```
   >
   >*每個片的最大多遍數*&#x200B;值必須保持設定為1（請參閱&#x200B;*已知問題*）。

* 對[!DNL Server\Menu\Admin]資料夾中的檔案所做的更改。

   * 移除查閱。
   * 新增封存檔案和使用者檔案。
   * 修改的檔案、導出檔案、日誌檔案。

   更新的檔案包括：

   ```
   Base\Menu\Admin\Order.txt (changes to menu)
   ```

   ```
   Base\Menu\Admin\Files.vw (removed Lookups and Events string, added 
   Archive and Users strings)
   ```

   ```
   Base\Menu\Admin\Archive Files.vw
   ```

   ```
   Base\Menu\Admin\User Files.vw
   ```

* 對[!DNL Base\Context\ServerDetails]資料夾中的檔案所做的更改：

   ```
   Base\Context\ServerDetails\Archive Files.vw
   ```

   ```
   Base\Context\ServerDetails\Export Files.vw
   ```

   ```
   Base\Context\ServerDetails\Log Files.vw
   ```

   ```
   Base\Context\ServerDetails\User Files.vw
   ```

   ```
   Base\Context\serverdetails\files.vw
   ```

   ```
   Base\workspaces\admin\dataset and profile\server files.vw
   ```

* 對成員訪問](/help/home/c-release-notes-insight/c-6-5.md)的自我管理所做的更改。[

   *Base*&#x200B;和&#x200B;*Server*&#x200B;包中的新檔案包括管理員要更新以訪問成員的[!DNL User List.cfg]。

## 修正的錯誤 {#section-3c47f9177e9d40d8938a5710399dc88f}

以下是在Data Workbench6.5中進行的顯著修正(自Data Workbench6.4發行以來)。

* 修正6.4版中導入的區塊解碼錯誤。處理超大訪客時發生問題。
* 修正編譯器升級導致的對齊錯誤。 在6.4版中受影響的復寫問題

## 已知問題 {#section-350ae121ca3442babbef4d0a2b027df1}

* *Windows憑證存放區*&#x200B;的使用僅限英文（不支援中文或日文）。
* *回歸* 分析圖表無法識別新的工作區選取項目。可能的因應辦法是在進行新選取後重做回歸。
* *每個片的最大多次傳遞*&#x200B;值必須保持設定為1。 在單一工作區中，您不能有超過一個以報表伺服器為目標的視覺效果：傾向分數、最佳配適歸因、關聯矩陣、關聯弦、關聯矩陣或關聯弦。
