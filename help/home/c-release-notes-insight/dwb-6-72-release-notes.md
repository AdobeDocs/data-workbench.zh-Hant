## 修正 {#section-160baf6ea04c45a993777ee4260691ed}

* 修正使用者使用使用者名稱和密碼登入時，無法使用「歸因」的工作站問題。
* 修正伺服器中AUTHENTICATE稽核記錄檔中遺失電子郵件的問題。
* 修正使用憑證和IP位址時，查詢記錄檔檔案名稱中遺失用戶端資訊的伺服器問題。
