# 處理離線 Sensor{#dealing-with-offline-sensors}

**建議頻率：** 視需要

這些情況是Web伺服器從其他伺服器中輪流退出，而Web伺服器因故障而完全關閉。

第一個要瞭解的概念是「截止」時間 [!DNL data workbench server]。
