# 感測器概述{#sensor-overview}

感測器是輕量型、低影響力的資料擷取軟體應用程式，可靠、安全且持續地從網路、應用程式和資料收集伺服器收集事件資料，以傳送至任何資料工作台伺服器。

[!DNL Sensor’s] 此工具可確保對您網際網路通道中發生的事件進行一致精確的測量。
