# 感測器概述{#sensor-overview}

Sensor是輕量型、低影響的資料擷取軟體應用程式，能可靠、安全且持續地從Web、應用程式和資料收集伺服器收集事件資料，以傳送至任何Data Workbench伺服器。

[!DNL Sensor’s] 檢測工具可確保對網際網路通道中發生的事件進行一致的準確測量。
