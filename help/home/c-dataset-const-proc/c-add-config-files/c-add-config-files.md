# 其他組態檔{#additional-configuration-files}

資料集設定檔或其繼承的設定檔的資料集目錄中包含的其他部分組態檔相關資訊。

雖然定義資料集設定檔時不一定需要這些檔案，但您可以編輯這些檔案，以指定其他參數來處理資料。
