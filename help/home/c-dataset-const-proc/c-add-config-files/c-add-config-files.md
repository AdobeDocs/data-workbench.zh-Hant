# 其他配置檔案{#additional-configuration-files}

資料集描述檔或其繼承的描述檔的資料集目錄中包含的部份其他設定檔案相關資訊。

雖然定義資料集描述檔時不一定需要這些檔案，但您可以編輯這些檔案，以指定其他參數來處理資料。
