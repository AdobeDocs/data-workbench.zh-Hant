# 標準轉換{#standard-transformations}

標準轉換採用一組輸入，執行一些操作，然後提供該操作以生成一個或多個輸出欄位。

以下各節提供了各標準轉換的說明、可用參數的表以及示例。 每個資料工作台伺服器DPU都提供所有標準轉換。
